## 你好，我是 Elysia

一名仍在学习和成长的大学生，也是一个喜欢把想法变成实际作品的人。

我对技术的兴趣并不来自某份完整的学习路线，而是来自一个又一个具体的问题：怎样让自己的服务器更稳定？一个游戏里的想法能不能真正做成 Mod？遇到故障时，如何找到原因并让它不再发生？

这些问题把我带到了 VPS、自建服务、网络安全和游戏开发。很多时候，我都是边做边学：先让它运行起来，再理解背后的原理，最后把过程整理成以后还能看懂的记录。

## 我在做什么

目前，我主要关注几个方向：

- **服务器实践**：部署和维护个人 VPS，研究安全加固、备份、监控与自动化运维。
- **游戏开发**：创作 WorldBox「逐鹿」Mod，把脑海中的世界观和玩法一点点实现出来。
- **技术记录**：整理真实的排障过程，包括错误判断、失败方案和最终解决办法。
- **生活观察**：记录技术之外，值得被记住的片段和思考。

我不希望只展示一个漂亮的结果。相比“照着命令复制就能成功”，我更愿意解释为什么这样做、哪里可能失败，以及我曾经踩过什么坑。

## 关于这个博客

OttoElysia 是我的个人实验场，也是一份公开的成长档案。

这里的内容未必都是标准答案，但会尽量做到真实、清楚、可以复现。旧文章也可能随着实践继续修订，因为技术会变化，我的理解也会变化。

博客使用 Hugo 生成，通过 GitHub 管理内容，并发布在 Cloudflare Pages。它没有复杂的后台，也不依赖自己的服务器运行——简单、快速，同时让我能够把更多注意力放在内容本身。
